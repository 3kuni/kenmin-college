## 1. 入学手続きに関して

### 1.1 手帳受け取り項目の削除
- **要望**: 手帳の受け取りの有無についての項目を削除する
- **理由**: スマホを利用した入学者には手帳が不要なため
- **対応状況**: 未対応

### 1.2 入学確認方法の簡素化
- **要望**: 担当者が毎日チェックしなくても済む、入学を確認する簡易な方法が欲しい
- **理由**: 現状では担当者が毎日チェックする必要が生じる
- **対応状況**: 未対応
